
## Hak Ediş Rule Engine

### Core Principle
Hak ediş is based on OPERATIONAL VALUE, not expense name.

- MALZEME_PLUS_IMALAT
- PURE_MALZEME
- NON_IMALAT

### Hak Ediş Policies
- ALWAYS_INCLUDED
- ALWAYS_EXCLUDED
- CONDITIONAL

### Formula
hak_edis_amount = amount * 0.07 (only if eligible)
